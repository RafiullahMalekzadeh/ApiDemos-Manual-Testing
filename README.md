# ApiDemos Mobile App Manual Testing Project

## Overview
This project includes manual testing of the ApiDemos Android application using BrowserStack App Live. Testing focuses on functional, UI/UX, and performance verification.

## Tools Used
- BrowserStack App Live
- ApiDemos-debug.apk
- Android Device: Pixel 7
- Manual Test Documentation

## Test Artifacts
###  Test Plan
Includes scope, objectives, environment, and methodology.

###  Test Cases
1. Functional – Open Views menu 
2. Functional – Buttons screen loads 
3. UI – Button text visibility 
4. UI – Alignment of Views list items 
5. Performance – App launch time 

## How to Run the App
1. Upload APK to BrowserStack App Live 
2. Select Android device 
3. Launch and test the application
